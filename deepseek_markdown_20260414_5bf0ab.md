# 提示词：生成网址收藏项目的前端需求文档

你是一位资深产品经理兼前端架构师。请根据以下要求，生成一份详细、精美、可直接用于开发的前端需求文档（PRD）。项目为“云藏·智能收藏夹”，后端使用 FastAPI + MySQL + Redis，前端使用 Vue3 + Element Plus / Ant Design Vue。

## 一、项目概述
- 产品定位：个人与社区结合的网址收藏工具，支持私有收藏、公开分享、官方推荐、数据统计及后台合规管理。
- 核心用户：普通用户、管理员。
- 设计风格：现代化、毛玻璃效果、暗色/亮色主题自适应、响应式布局（PC 优先，兼顾平板）。

## 二、页面清单及详细需求

### 1. 导航页（首页）
**路由**：`/`
**布局**：
- 顶部导航栏：Logo（云藏）、搜索框（全局搜索网址/标题/标签）、主题切换、用户头像下拉菜单（登录/注册/个人中心）。
- Hero 区域：大标题“收藏不止于链接，更有洞察”、Chip 式热门标签云。
- 内容区：三栏瀑布流布局
  - 左侧：官方精选（固定 3 条，带小编推荐语）
  - 中间：最新公开分享（按时间倒序，卡片展示标题、域名、点赞数）
  - 右侧：热门排行（本周点击量 Top 10，显示排名、标题、所属分类）
- 底部：简洁版权信息。

**交互**：
- 未登录用户可浏览所有公开内容，点击收藏详情需跳转登录。
- 搜索框支持实时联想（防抖 300ms）。

### 2. 收藏页（我的收藏）
**路由**：`/my/bookmarks`
**权限**：仅登录用户
**布局**：
- 左侧边栏（可折叠）：
  - 全部收藏（数量）
  - 分类列表（支持编辑、删除分类）
  - 标签云（按使用频率显示大小）
- 右侧主区域：
  - 顶部操作栏：新增收藏按钮（弹窗表单）、批量删除、导入/导出（HTML 书签）、筛选（按分类/标签/日期）。
  - 卡片列表（网格/列表切换）：
    - 卡片内容：网站图标（favicon）、标题、网址、简短描述、标签组、操作按钮（编辑、删除、移动分类、分享开关）。
    - 每个卡片右下角显示“点击次数”及“最后访问时间”。
  - 分页器（支持每页 20/50/100）。

**特殊功能**：
- 智能标签推荐：添加收藏时，后端调用 LLM 自动生成标签，前端展示推荐标签（带“采纳”按钮）。
- 拖拽排序：支持拖拽卡片调整顺序（本地存储排序偏好）。

### 3. 官方分享页面
**路由**：`/official`
**说明**：展示由管理员精选并审核通过的优质公开分享，也可包含官方整理的资源专题。
**布局**：
- Banner 轮播图（官方推荐专题，点击跳转专题详情）。
- 分类标签栏（技术、设计、生活、工具……），高亮当前选中。
- 卡片墙（3 列，响应式）：
  - 每个卡片包含：网站截图（自动生成或默认）、标题、简短介绍、收藏人数、点赞按钮。
  - 卡片底部显示“官方推荐”徽章。
- 右侧悬浮工具栏：回到顶部、反馈按钮。

**交互**：
- 点赞需要登录，未登录跳转弹窗。
- 点击卡片进入详情页（展示完整 URL、详细说明、相关推荐）。

### 4. 统计页面
**路由**：`/stats`
**权限**：登录用户
**布局**：
- 顶部日期范围选择器（本周/本月/自定义）。
- 仪表盘式布局（卡片 + ECharts 图表）：
  - **我的点击热力图**：日历热力图，展示每日点击收藏的次数。
  - **分类占比饼图**：根据收藏数量展示各分类占比。
  - **热门网址排行榜**（柱状图）：展示自己收藏中点击次数最多的前 10 个网址。
  - **时段分布折线图**：一天 24 小时的点击频率（用于分析使用习惯）。
- 数据表格：所有收藏的详细点击记录（网址、最后点击时间、总点击次数），支持导出 CSV。

**额外**：
- 提供“分享统计报告”按钮，生成图片版统计摘要（可选）。

### 5. 个人中心
**路由**：`/profile`
**权限**：登录用户
**布局**：左右两栏
- 左侧垂直菜单：个人信息、账号安全、我的分享、通知设置、主题偏好。
- 右侧内容区：
  - **个人信息**：头像上传（支持裁剪）、昵称、个人简介（Markdown 预览）、邮箱绑定。
  - **账号安全**：修改密码、绑定手机、两步验证（QR 码）。
  - **我的分享**：列出所有我公开分享的收藏，支持取消分享、查看访问统计。
  - **通知设置**：邮件/站内信开关（收藏被点赞、评论等）。
  - **主题偏好**：深色/浅色/跟随系统。

### 6. 管理员单独页面
**路由前缀**：`/admin`
**权限**：仅 admin 角色
**布局**：侧栏导航 + 顶部面包屑
**包含子页面**：

#### 6.1 分享合规审核
**路由**：`/admin/audit`
**功能**：展示所有用户公开分享的网址列表，标记“待审核/已通过/违规”。
- 表格列：ID、用户昵称、网址标题、URL、提交时间、当前状态。
- 操作按钮：通过、驳回（填写原因）、加入黑名单（禁止该用户分享）。
- 预览功能：点击网址可在 iframe 弹窗中预览（需安全沙箱）。
- 批量审核：勾选多条后批量通过。

#### 6.2 违法网址管理
**路由**：`/admin/blacklist`
**功能**：维护黑名单域名或关键词。
- 可新增/删除黑名单规则（域名精确匹配、正则表达式）。
- 展示命中黑名单的网址列表（谁分享的、何时命中）。
- 一键清理所有违规分享，并通知相关用户。

#### 6.3 官方推荐管理
**路由**：`/admin/official`
**功能**：从已审核通过的公开分享中选择加入“官方分享页面”。
- 卡片列表（同用户侧），每个卡片有一个“设为官方推荐”开关。
- 可设置推荐权重（决定轮播图顺序）。
- 编辑推荐语。

#### 6.4 系统统计
**路由**：`/admin/stats`
**功能**：全局仪表盘
- 总用户数、总收藏数、总公开分享数、今日活跃用户。
- 违规趋势折线图（近 30 天违规数量）。
- 热门分类统计（所有用户）。

## 三、通用前端要求
1. **路由守卫**：未登录不可访问个人中心、收藏页、统计页；非管理员访问 admin 页面跳转 403。
2. **全局状态**：使用 Pinia 存储用户信息、主题、通知未读数。
3. **API 交互**：封装 axios，支持 JWT 自动注入、401 跳转登录、请求防抖。
4. **错误处理**：统一 Toast 提示（成功/错误/警告），网络错误友好提示。
5. **响应式**：PC 宽屏（>1400px）显示 3 列卡片，平板（768-1400px）显示 2 列，手机显示 1 列，侧边栏自动隐藏为抽屉。
6. **性能**：列表虚拟滚动（当收藏超过 200 条），图片懒加载。

## 四、非功能性需求
- 页面加载首屏时间 < 1.5 秒（LCP）。
- 支持 PWA（可选，作为 bonus）。
- 页面需包含优雅的加载骨架屏和空状态插画。

请输出一份结构清晰的 Markdown 文档，包含：
