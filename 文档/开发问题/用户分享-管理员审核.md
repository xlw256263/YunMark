#### 问题 2：数据库表创建失败

**症状**：迁移脚本报错

**解决**：
```sql

SHOW TABLES LIKE 'bookmark_shares';

-- 如果表不存在，手动创建
-- 或者删除表后重新运行迁移脚本

```

**解决**：

#### 问题 4：分享按钮点击无反应

**症状**：点击书签的"分享"按钮没有弹窗或提示

**解决**：
1. 打开浏览器开发者工具（F12）
2. 查看 Console 是否有 JavaScript 错误
3. 查看 Network 标签，确认 API 请求是否发送
4. 检查是否已登录（分享功能需要登录）

### 6. API 快速测试（使用 PowerShell）

```powershell
# 设置变量
$TOKEN = "your_jwt_token_here"
$BASE_URL = "http://localhost:8000/api/v1"
$HEADERS = @{ Authorization = "Bearer $TOKEN" }

# 1. 创建分享
$body1 = @{ bookmark_id = 1 } | ConvertTo-Json
Invoke-RestMethod -Uri "$BASE_URL/shares" -Method Post -Headers $HEADERS -Body $body1 -ContentType "application/json"

# 2. 获取我的分享列表
Invoke-RestMethod -Uri "$BASE_URL/shares/my-shares?page=1&page_size=10" -Method Get -Headers $HEADERS

# 3. 提交审核（假设 share_id = 1）
$body2 = @{ share_id = 1 } | ConvertTo-Json
Invoke-RestMethod -Uri "$BASE_URL/shares/submit" -Method Post -Headers $HEADERS -Body $body2 -ContentType "application/json"

# 4. 获取公开分享（无需 Token）
Invoke-RestMethod -Uri "$BASE_URL/public/shares?page=1&page_size=10&sort=latest" -Method Get
```
